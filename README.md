# From-Scratch-ML-Algorithms
From-scratch implementation of various machine learning algorithms in python.
## Contents
The algorithms are:
* Random Forests
* AdaBoost
* Gradient Boosting
* Newton-Raphson Algorithm for Classification
* Linear Regression
## Dependencies
Requires Python >= 3.8, Numpy >= 1.19, scikit-learn >= 0.23.
